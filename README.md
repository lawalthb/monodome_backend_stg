# monodomebackend

- To install clone the repo
```
 git clone https://github.com/talosmart/monodome_backend_users.git
```

- Run composer install
```
 composer install
```

- Run migration using
```
php artisan migrate:fresh --seed
```

Enjoy

NOTE: MAKE YOU SEED DEFAULT BECAUSE OF SITE SETTING.
